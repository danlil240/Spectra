Spectra — WebGPU Demo

GPU-accelerated scientific plotting in the browser

Initializing WebGPU…

This page runs the Spectra C++20 plotting library compiled to WebAssembly via Emscripten. The rendering uses the WebGPU API — no Vulkan, no OpenGL, no software fallback. Requires a WebGPU-capable browser (Chrome 113+, Edge 113+, or Firefox Nightly with dom.webgpu.enabled).
