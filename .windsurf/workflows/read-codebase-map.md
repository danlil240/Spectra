# Read Codebase Map

**This workflow MUST be executed before starting any task in the Spectra project.**

1. Read the file `CODEBASE_MAP.md` at the project root to understand:
   - Project architecture and directory structure
   - Class hierarchy and ownership model
   - Rendering pipeline and data flow
   - Multi-window system design
   - Key patterns and gotchas (deferred deletion, ImGui contexts, dirty flags)
   - File cross-reference for quick lookup
   - Naming conventions and coding standards

2. After reading, proceed with your assigned task using the architecture knowledge from the map.
