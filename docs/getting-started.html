Getting Started with Spectra

Set up Spectra for C++20 and Python in minutes.

1) Prerequisites

DependencyRequiredNotes
C++ compilerC++20GCC 13+, Clang 17+, or MSVC 2022+
Vulkan SDK1.2+GPU backend requirement
CMake3.20+Build and configure
Python3.9+For Python API usage

2) Build from Source

git clone https://github.com/danlil240/Spectra.git
cd Spectra
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j$(nproc)
Tip: for docs/testing workflows, keep SPECTRA_BUILD_TESTS=ON.

3) First Plot (C++)

#include <spectra/easy.hpp>
#include <vector>

int main() {
    std::vector<float> x{0.f, 1.f, 2.f, 3.f};
    std::vector<float> y{0.f, 1.f, 0.5f, 1.5f};

    spectra::plot(x, y, "c-o").label("demo");
    spectra::title("First Spectra Plot");
    spectra::show();
}

4) Python Quick Start

pip install spectra-plot

python -c "import spectra as sp; sp.plot([0,1,2],[0,1,0.5]); sp.show()"
For embed mode, build with -DSPECTRA_BUILD_EMBED_SHARED=ON.
